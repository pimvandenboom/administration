# Administratie Tool

Een snelle en eenvoudige tool voor het beheren van je administratie en kosten.

## Features

- 📝 Snel kosten invoeren
- 📊 Automatische verwerking en berekening
- 📈 Overzichtelijke rapportage
- 💾 Lokale opslag van je gegevens
- 📅 Categorie-indeling
- 📑 Export naar CSV/PDF

## Aan de slag

1. Clone deze repository
2. Open `index.html` in je browser
3. Start met het invoeren van je kosten

## Technologie

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage voor opslag
